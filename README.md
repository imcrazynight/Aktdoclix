# Aktdoclix
Ein einfaches Archivprogramm für Vereine, Gemeinde und Privatleute
Der Name Aktdoklix ist eine Mischung aus Akten,dokumentieren und meinen Vornamen Felix
Servus
In meiner Freizeit habe ich bekonnen Heimatkunde zu betreiben Nach über 5 Jahren und Digitalisierung   mit meinem Fotoabarat und neuerdings Buchscannern:) stand ich vor einem Problem, Chaos auf meine Computer.
Wer kleine Archive verwaltet, bekommt oft den Rat: "Nimm doch Excel" Doch wer mehr als 200 Signaturen hat,
merkt schnell Excel wird unübersichtlich.
Professionelle Archiv-Software ist für mich keine lösung es war unnötig teuer als privatperson.
Die Bedienung von Profi-Software ist oft unnötig kompliziert.
Ich habe versucht so variabel möglichzu machen wie es geht für jeden anpassbar zumachen.
